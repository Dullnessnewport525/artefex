## 🛠 Troubleshooting

### The file will not load
Check that the file is not renamed with the wrong extension. Try another image file to see if the app opens it.

### The output looks too soft
Try a different restore setting or use a less aggressive pass.

### The app runs slowly
Close other apps and try smaller files first.

### The result has odd marks
The file may be too damaged for a clean restore, or the source may already have heavy compression loss.

### The folder is empty after processing
Check the output path and confirm the app had permission to write files there.
